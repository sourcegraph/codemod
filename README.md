# @sourcegraph/codemod

A toolkit for writing codemods powered by [ts-morph](https://github.com/dsherret/ts-morph) and [PostCSS](https://github.com/postcss/postcss).

## Setup

```sh
yarn
```

## Use

```sh
yarn transform --help
```

## Other commands

```sh
yarn build
yarn build:clean
yarn build:watch

yarn test
yarn format
yarn format:check
yarn lint
```
